Web Server (abc Samples)
This is a simple web server with the following functions:
  1. Fetch static files
    GET /resources/<file>
    Streams down the chosen file. Other methods to /resources/... are not allowed.
  2. Make REST requests
    GET | POST | PUT | DELETE <resource>
    Returns a JSON response that wraps the original request.
Examples:
  • GET /resources/index.html
  • POST /people
    Content-Type: application/json
    Accept: application/json
    {
        "id": 123456,
        "name": "John Doe",
        "birthYear": 1980
    }
